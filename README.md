# 安裝NeoBundle

```bash
curl https://raw.githubusercontent.com/Shougo/neobundle.vim/master/bin/install.sh | sh
```

# 設定vimrc

```bash
ln -fs $PWD/vimrc ~/.vimrc
```

# 設定bash prompt


```bash
ln -fs $PWD/bash_prompt.sh ~/.bash_prompt
echo '. ~/.bash_prompt' >> ~/.bashrc
```
